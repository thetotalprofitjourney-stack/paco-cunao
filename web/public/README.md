# Archivos Estáticos

Este directorio contiene los archivos estáticos que se sirven públicamente en la aplicación web.

## Imagen del Hero

**IMPORTANTE**: Sube aquí la imagen generada del Hotel Villa Carmen con el nombre:

```
hero_paco.webp
```

Esta imagen se usará como fondo de pantalla completa en la landing page.

### Especificaciones recomendadas:
- **Formato**: WebP (mejor compresión)
- **Dimensiones**: 1920x1080px o superior (16:9)
- **Peso**: < 500KB optimizado
- **Calidad**: Alta, se verá a pantalla completa

### Ubicación esperada:
```
/web/public/hero_paco.webp
```

### URL pública:
La imagen estará disponible en: `http://localhost:3001/hero_paco.webp`
